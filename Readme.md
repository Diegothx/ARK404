## Getting Started
1. Environment Variables

Create a .env file at the root of the project:
```
# Database
DB_USER=testuser
DB_PASSWORD=testpass
DB_NAME=testdb
DB_HOST=db
DB_PORT=5432

# Backend
BACKEND_PORT=5000

# Frontend
FRONTEND_PORT=3000
```

2. Development Mode (With Hot Reload) 

Start dev environment:
`make dev`

Generate frontend API client:
`make make generate-frontend-api`

Create migration:
`make makemigration m="add guestbook"`

Apply migrations:
`make migrate`

Stop containers:
`make dev-down`

Delete frontend API client:
`make clean-frontend-api`
